# Markdown Slides Static

Pure static frontend. No npm, no backend, no build step.

## Files

- `index.html`
- `styles.css`
- `app.js`

## Usage

Open `index.html` directly in a browser, or deploy the folder to any static web hosting.

## Features

- Paste Markdown
- Split slides by `---`
- Source / Preview switch
- Markdown quick toolbar
- Slide preview
- Presentation mode
- Export PDF through browser print
